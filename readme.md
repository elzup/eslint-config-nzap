# eslint-config-nzap


## Usage

```
npm i -D eslint-config-nzap
```


## Usage

.eslintrc

```
{
  "extends": "nzap"
}
```

### React

```
{
  "extends": ["nzap", "nzap/react"]
}
```

### TypeScript

```
{
  "extends": ["nzap", "nzap/typescript"]
}
```

### jest

```
{
  "extends": ["nzap", "nzap/jest"]
}
```
